Try to Escape

Description



Project “Try to escape” is written in Python File. When we start the game it will display the Title of the game, Tap to start and home icon on the top right corner of the screen. In home we will have options related to types of balls, types of paths, music, and controls and quit. When we click on the Tap to start, a ball moving in forward direction escapes from obstacles by moving left or right that come in the middle of the path. As the ball moves forward the speed of the ball will increase for every 30 seconds. As time increases the number of obstacles will increase. For every second score will be increased by 1 point. If the ball hits any obstacle, the game ends and it will display the score, restart, quit from the game and display the starting page. If you want to quit the game click on the home icon and quit the game.



Technologies used: Python
Size of the team: 3
